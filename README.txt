# 16-to-10-number-translate

1) Откройте приложение (.exe формат)
2) Введите число в 16-й системе исчисления размером до 15 знаков
3) Получите это число в 10-й системе исчисления

В .cpp файле исходный код

(!) Скорее всего, антивирус не пропустит исполняемый файл, в таком случае выключите антивирус и проигнорируйте предупреждение
